Last Update 1/7/2020. 
This Matlab code package recreates key figures from the manuscript:
Novick K, Katul G (under review) Reforestation and Duality Between Surface and Air Temperature. 
Code authored by Dr. Kim Novick, Associate Professor, School of Public and Environmental Affairs, Indiana University, Bloomington, USA.
Begin by opening the “TATS_mainForSharing.m” file. All subroutines are called from this file. Extensive commenting appears in this file, and all subroutines.
